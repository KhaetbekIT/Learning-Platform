# Routing Structure for student portal

## Platform
- **/dashboard** - Главная панель
- **/courses** - Курсы
- **/chat** - Чат
  - **/chat/my-courses** - Мои курсы
  - **/chat/catalog** - Каталог
  - **/chat/details** - Детали
- **/live** - Прямой эфир
  - **/live/calendar** - Календарь событий
  - **/live/live-chat** - Live-чат
